线性回归与模型诊断

本练习使用 MASS::Melanoma：以连续结局 thickness 为例，练习 lm() 拟合、系数解释、区间估计与基础诊断（残差图、影响点）。
学习目标
会拟合简单与多重线性回归；能解释系数与置信区间；能用诊断图发现常见问题并给出处理思路

0. 载入 Melanoma

0.1 载入数据

options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))
if (!requireNamespace("MASS", quietly = TRUE)) install.packages("MASS")
library(MASS)
data(Melanoma)
mel = as.data.frame(Melanoma)

1. 简单线性回归

1.1 拟合与系数解释

模型：用 age 解释 thickness，形式为 thickness ~ age。
Step 1 / 拟合与查看摘要
fit1 = lm(thickness ~ age, data = mel)
summary(fit1)
Step 2 / 散点图 + 回归线
plot(mel$age, mel$thickness,
     xlab = "age", ylab = "thickness",
     main = "thickness ~ age",
     pch = 19, col = rgb(13/255, 148/255, 136/255, 0.35))
abline(fit1, col = "#b91c1c", lwd = 2)

1.2 置信区间与预测

Step 3 / 系数置信区间
confint(fit1)
Step 4 / 给定年龄的预测（点预测）
newd = data.frame(age = c(30, 50, 70))
predict(fit1, newdata = newd)

2. 多重线性回归与诊断

2.1 加入协变量与分类变量

示例：加入 ulcer 与 sex（二分类变量）以及 year。其中 sex 在原数据里编码为 0/1，按分类变量进入模型更符合解释习惯。
Step 5 / 多重线性回归
fit2 = lm(thickness ~ age + factor(sex) + ulcer + year, data = mel)
summary(fit2)
confint(fit2)

2.2 诊断图与影响点

Step 6 / 四张经典诊断图
par(mfrow = c(2, 2))
plot(fit2, which = 1:4)
par(mfrow = c(1, 1))
Step 7 / Cook 距离：查看影响点（数值最大的若干条）
ck = cooks.distance(fit2)
head(sort(ck, decreasing = TRUE), 8)
注意：影响点不等于“必须删除”。通常需要回到原始记录核对：是否录入问题、是否极端病例、是否提示模型形式需要调整（变换/加项/稳健方法）。
