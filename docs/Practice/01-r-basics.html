01 · R 语言数据类型与运算（练习）

第一部分 · 约 10 分钟

基础语法与变量

把 R 当作计算器；用 <- 赋值；区分数值、字符、逻辑；用 class()/typeof() 做类型检查。

  1. 计算：1 + 1、2 ^ 3；再试 10 / 3、10 %/% 3、10 %% 3，理解除法/整除/取余。
  2. 创建变量：my_age <- 25、my_name <- \"Data_Miner\"、is_student <- TRUE，并打印查看。
  3. 对三个变量分别运行 class() 与 typeof()，记录输出。

第二部分 · 约 20 分钟

向量 Vector

用 c() 创建向量；体验向量化计算；掌握 [] 提取元素（R 下标从 1 开始）。

  1. 创建 ages 与 names 向量。
  2. 运行 ages + 5，观察“对整列一起算”。
  3. 用 names[1] 与 names[2:4] 提取元素。

ages <- c(22, 25, 29, 31, 24)
names <- c("Alice", "Bob", "Charlie", "David", "Eve")

ages + 5

第三部分 · 约 45 分钟

数据框 Data Frame

用 data.frame() 拼装表；掌握拿到数据必做的“看一眼四件套”：View/head/str/summary；用 $ 与 [行, 列] 提取。

  1. 拼装 my_data，包含 Name、Age、Score 三列。
  2. 依次运行：View(my_data)、head(my_data, 3)、str(my_data)、summary(my_data)。
  3. 用 my_data$Age 提取列；用 mean(my_data$Score) 计算平均分。
  4. 练习二维索引：my_data[1, 2]、my_data[1, ]、my_data[, \"Name\"]。

my_data <- data.frame(
  Name = names,
  Age = ages,
  Score = c(88, 92, 75, 95, 80)
)

str(my_data)

拓展

字符、因子、NA 与容器

字符处理、因子分类、缺失值处理，以及三类常用容器（vector / list / data.frame）。

字符与文本

  1. 用 paste/paste0 拼接患者标签（如 \"P012\"）。
  2. 对 \"黄芪\"、\"炙甘草\" 用 nchar() 看字符长度。
参考答案 
paste("患者", 12, sep = "-")
paste0("P", sprintf("%03d", 12))
nchar("黄芪")
nchar("炙甘草")

因子与分类

  1. 把证型向量转为 factor，并用 levels/table 查看水平与频数。
  2. 指定 levels 固定显示顺序。
参考答案 
syndrome <- factor(c("气虚", "阴虚", "气虚", "气虚", "阴虚"))
levels(syndrome)
table(syndrome)
syndrome2 <- factor(c("气虚", "阴虚", "气虚", "气虚", "阴虚"),
                    levels = c("气虚", "阴虚"))
table(syndrome2)

缺失值 NA

  1. 构造 x <- c(10, NA, 30)，比较 mean(x) 与 mean(x, na.rm = TRUE)。
  2. 用 is.na 与 sum 统计缺失数。
参考答案 
x <- c(10, NA, 30)
mean(x)
mean(x, na.rm = TRUE)
sum(is.na(x))
mean(x[!is.na(x)])

容器：vector / list / data.frame

  1. 创建一个向量，一个 list（把不同类型放一起），以及一个 data.frame（同长度列组成表）。
  2. 分别用 str() 查看三者结构，写一句话描述差异。
参考答案 
v <- c(1, 2, 3)
lst <- list(name = "李某", age = 25, labs = c(1.2, 3.4))
df <- data.frame(id = 1:3, sbp = c(120, 118, 135))
str(v); str(lst); str(df)
