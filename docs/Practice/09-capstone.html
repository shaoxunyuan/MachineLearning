综合练习：从数据到报告（练习）

本页把前面章节练习的核心环节串起来：载入数据 → 描述统计 → 分组比较 → 回归建模 → 整理输出。数据仍使用 MASS::Melanoma，以便专注流程与表达。
学习目标
把一段分析写成可复现脚本；能产出 1 张图 + 1 张表 + 1 段结论文字

0. 载入数据

options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))
if (!requireNamespace("MASS", quietly = TRUE)) install.packages("MASS")
library(MASS)
data(Melanoma)
mel = as.data.frame(Melanoma)
str(mel)

1. 描述统计与图表

Step 1 / 单变量描述：thickness、age
summary(mel$thickness)
summary(mel$age)
Step 2 / 分组描述：按 ulcer 分组的 thickness
by(mel$thickness, mel$ulcer, summary)
Step 3 / 作图：箱线图
boxplot(thickness ~ factor(ulcer), data = mel,
        xlab = "ulcer (0/1)", ylab = "thickness",
        main = "thickness by ulcer")

2. 推断：分组比较

Step 4 / 两组比较：Welch t 与 Wilcoxon
t_res = t.test(thickness ~ ulcer, data = mel)
w_res = wilcox.test(thickness ~ ulcer, data = mel)

t_res$p.value
w_res$p.value

3. 回归建模：线性 vs Logistic

Step 5 / 线性回归：thickness 作为连续结局
fit_lm = lm(thickness ~ age + factor(sex) + ulcer + year, data = mel)
summary(fit_lm)
Step 6 / Logistic 回归：death 作为二分类结局
mel$death = ifelse(mel$status == 1, 1, 0)
fit_glm = glm(death ~ age + thickness + ulcer + factor(sex),
              data = mel, family = binomial)
summary(fit_glm)

# OR 与 95% CI
exp(cbind(OR = coef(fit_glm), confint(fit_glm)))

4. 结果整理为“可报告”表格

目标：把模型输出整理成 1 张简洁表格（变量、估计值、区间、p 值），便于写进实验报告。
Step 7 / 线性回归：系数表
co = summary(fit_lm)$coef
out_lm = data.frame(
  term = rownames(co),
  estimate = co[, "Estimate"],
  se = co[, "Std. Error"],
  p = co[, "Pr(>|t|)"],
  row.names = NULL
)
head(out_lm)
Step 8 / Logistic：OR 表
co2 = summary(fit_glm)$coef
ci2 = confint(fit_glm)
out_glm = data.frame(
  term = rownames(co2),
  OR = exp(co2[, "Estimate"]),
  lo = exp(ci2[, 1]),
  hi = exp(ci2[, 2]),
  p = co2[, "Pr(>|z|)"],
  row.names = NULL
)
head(out_glm)
