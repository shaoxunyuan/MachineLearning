练习 · 01 R 基础入门

本页按「计算器 → 变量 → 向量 → 数据框 → 函数与扩展包」顺序编排，时间供自学参考。请在 RStudio 中 File → New File → R Script 新建脚本，逐段运行（选中代码按 Ctrl+Enter 或点击 Run）。细节与延伸见第 01 章讲义。每部分末有「参考答案」折叠区。

Part 1 · 约 10 分钟

基础语法与变量

把 R 当作计算器；用 <- 赋值；区分数值、字符、逻辑；习惯用 # 写注释。

  1. 在控制台或脚本中计算：1 + 1、2 ^ 3（幂运算）、10 / 3 与 10 %/% 3、10 %% 3，观察整除与取余。
  2. 用 <- 创建：my_age（数值）、my_name（字符，需引号）、is_student（逻辑，TRUE/FALSE 全大写）。在下一行直接输入变量名或 print() 查看结果。
  3. 对三个变量分别调用 class() 与 typeof()，记录数值型在 R 中的常见类型名。
  4. （想一想）赋值用 <- 与 = 在多数场景可互换，但本课与社区习惯以 <- 为主；避免在复杂表达式里混用造成阅读困难。

Part 2 · 约 20 分钟

向量 Vector

用 c() 把多个值收成向量；体验向量化运算；用 [] 按位置取元素。注意：R 下标从 1 开始。

  1. 创建 ages <- c(22, 25, 29, 31, 24)，计算 ages + 5（每人加 5 岁）。
  2. 创建 names <- c("Alice", "Bob", "Charlie", "David", "Eve")，取出第 1 个名字、第 2～4 个名字。
  3. 用 length(ages) 看长度；试 seq(1, 10, by = 2) 与 rep(0, 5)。
  4. （医学小例子）设五次体温 temp <- c(36.5, 37.1, 36.8, 37.4, 36.6)，用 temp > 37 得到是否「高于 37」的逻辑向量，再用 temp[temp > 37] 取出具体数值。

ages <- c(22, 25, 29, 31, 24)
names <- c("Alice", "Bob", "Charlie", "David", "Eve")

Part 3 · 约 45 分钟

数据框 Data Frame

Excel/CSV 读入后多为数据框；掌握拼装、初探、$ 取列与 [行, 列] 索引。

  1. 用上方 names、ages 拼装：my_data <- data.frame(Name = names, Age = ages, Score = c(88, 92, 75, 95, 80))（若未运行上一节，请先在本脚本里重新定义 ages、names）。
  2. 依次运行：View(my_data)、head(my_data, 3)、str(my_data)、summary(my_data)，用自己的话写下每个函数适合在什么场景使用。
  3. 用 my_data$Age、mean(my_data$Score) 取列与聚合。
  4. 用二维索引：my_data[1, 2]、my_data[1, ]、my_data[, "Name"]；再试 my_data[2, , drop = FALSE] 与 my_data[2, ] 在控制台显示形态上的区别。
  5. （拓展）用 subset(my_data, Age >= 25) 筛选年龄不小于 25 的行。

函数与 R 包机制

调用内置函数；理解 install.packages() 与 library() 的分工；用扩展包自带数据集练手。

  1. 对 my_data$Score 求 max()、min()、range()。
  2. 首次使用需安装：install.packages("palmerpenguins")（只需做一次；若下载慢，见第五部分「CRAN 镜像」）。安装后每次新开会话要 library(palmerpenguins)。
  3. 加载成功后运行 head(penguins)、str(penguins)；用 mean() 对某一数值列计算均值（注意该列是否有 NA，必要时 na.rm = TRUE）。
  4. 若暂时无法安装扩展包，可用内置 iris 代替：head(iris)、mean(iris$Sepal.Length)。

拓展

RStudio 配置与工作目录

减少「包装不上、脚本找不到文件」类问题，与课堂环境对齐。

  1. CRAN 镜像：Tools → Global Options → Packages，在 CRAN mirror 中选国内镜像（如清华 https://mirrors.tuna.tsinghua.edu.cn/CRAN/），再重试 install.packages()。
  2. 工作目录：Session → Set Working Directory → Choose Directory... 选到你的项目文件夹；或在脚本首行使用 setwd("你的路径")（路径中含中文时尤其注意引号与斜杠）。控制台运行 getwd() 确认当前目录。
  3. 脚本与运行：左上面板保存为 .R；选中多行后 Ctrl+Enter 逐段送入控制台。右下角 Files 可点开数据文件；Environment 查看已创建的 my_data 等对象。
  4. （可选）在控制台执行 .libPaths()，了解扩展包安装位置；一般无需改，除非磁盘空间或权限特殊。
