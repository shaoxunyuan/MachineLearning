Exercise 01

变量与类型检查

巩固 <- 赋值与 class()、typeof()、str() 的用法。

  1. 创建三个变量：收缩压数值 bp_sys <- 128、患者姓名字符 patient_name <- "李某"、是否急诊逻辑 is_emergency <- FALSE。
  2. 分别对三者调用 class() 与 typeof()，观察输出差异（尤其注意数值型在 R 中的存储名称）。
  3. 对 bp_sys 再调用 str()，说明控制台打印了哪些信息。
Starter（可复制到脚本中补全）
bp_sys <- 128
patient_name <- "李某"
is_emergency <- FALSE

# 在此续写 class / typeof / str
参考答案 
class(bp_sys); typeof(bp_sys)       # "numeric", "double"
class(patient_name); typeof(patient_name)  # "character", "character"
class(is_emergency); typeof(is_emergency)  # "logical", "logical"
str(bp_sys)
Exercise 02

数值：剂量拆分与取余

练习除法、整除 %/% 与取余 %%，对应讲义中的分次给药与疗程天数场景。

  1. 某味药一日总量 total_g <- 45，分 n_times <- 3 次服用，计算每次剂量（克）。
  2. 疗程以 7 天为周期，总观察天数 days <- 20：用 %/% 求完整周期个数，用 %% 求剩余天数。
  3. 用 1L 与 1 各赋给一个变量，比较 class() 结果。
Starter
total_g <- 45
n_times <- 3
# each_g <- ?

days <- 20
cycle <- 7
# full_cycles <- ?
# remainder <- ?
参考答案 
each_g <- total_g / n_times        # 15
full_cycles <- days %/% cycle      # 2
remainder <- days %% cycle         # 6
class(1L)   # "integer"
class(1)    # "numeric"
Exercise 03

逻辑与向量化比较

理解向量上比较运算与 &、| 的向量化含义，区分 &&。

  1. 定义 sbp <- c(118, 135, 128, 142)，生成逻辑向量「收缩压是否大于 130」。
  2. 用该逻辑向量对 sbp 做子集选取，只保留大于 130 的值。
  3. 在控制台分别执行 TRUE & FALSE 与 TRUE && FALSE，再尝试 c(TRUE,FALSE) & c(FALSE,TRUE) 与 c(TRUE,FALSE) && c(FALSE,TRUE)，说明后者为何只返回一个结果。
Starter
sbp <- c(118, 135, 128, 142)
# hi <- sbp ??? 130
# sbp[???]
参考答案 
hi <- sbp > 130
sbp[hi]
# c(TRUE,FALSE) & c(FALSE,TRUE)   # 向量逐元素
# c(TRUE,FALSE) && c(FALSE,TRUE)    # 只对第一个元素求值后短路，返回单个 logical

Exercise 05

因子与频数表

将分类变量存为 factor，查看 levels 与 table。

  1. 对向量 c("气虚", "阴虚", "气虚", "气虚", "阴虚") 构造因子 syndrome，默认按字母序水平。
  2. 输出 levels(syndrome) 与 table(syndrome)。
  3. （拓展）使用 factor(..., levels = c("气虚", "阴虚")) 固定水平顺序，再查看 table 是否变化。
参考答案 
syndrome <- factor(c("气虚", "阴虚", "气虚", "气虚", "阴虚"))
levels(syndrome)
table(syndrome)
syndrome2 <- factor(c("气虚", "阴虚", "气虚", "气虚", "阴虚"),
                    levels = c("气虚", "阴虚"))
table(syndrome2)
Exercise 06

缺失值 NA

理解 NA 在运算中的传播及 na.rm、is.na。

  1. 对 x <- c(10, NA, 30) 计算 mean(x)，观察结果；再计算 mean(x, na.rm = TRUE)。
  2. 用 is.na(x) 得到逻辑向量，并用 sum(is.na(x)) 统计缺失个数。
  3. 用 x[!is.na(x)] 去掉缺失后求均值，与上一步 na.rm 结果对照。
参考答案 
x <- c(10, NA, 30)
mean(x)              # NA
mean(x, na.rm = TRUE) # 20
sum(is.na(x))        # 1
mean(x[!is.na(x)])   # 20
Exercise 07

data.frame 基础

构造小型「患者表」，练习按列名提取与子集。

  1. 创建 dat <- data.frame(id = 1:3, sbp = c(120, 118, 135), rx = c("方剂A", "方剂B", "方剂A"))。
  2. 取出第二行（三种索引方式任选其一：[2, ]、dat[2, , drop = FALSE] 等）。
  3. 用 dat$sbp 取出收缩压列；用 subset(dat, sbp < 130) 筛选收缩压低于 130 的行。
Starter
dat <- data.frame(
  id = 1:3,
  sbp = c(120, 118, 135),
  rx = c("方剂A", "方剂B", "方剂A")
)
# 续写子集与 subset
参考答案 
dat[2, , drop = FALSE]
dat$sbp
subset(dat, sbp < 130)
