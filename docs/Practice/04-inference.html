统计推断基础

本练习使用 MASS 包自带的 Melanoma 数据集，练习连续变量与分类变量的常用统计推断：置信区间、两组比较（Welch / Wilcoxon）、列联表、卡方检验与 Fisher 精确检验。
学习目标
会计算均值置信区间；会选择两组比较方法并解释结果；会做列联表检验并给出简要解读

0. 载入 Melanoma

0.1 载入数据

Step 1 / 安装并加载 MASS
options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))
if (!requireNamespace("MASS", quietly = TRUE)) install.packages("MASS")
library(MASS)
Step 2 / 载入数据并定义主表 mel
data(Melanoma)
mel = as.data.frame(Melanoma)
str(mel)
结果：对象 mel 可用于后续推断练习。

1. 连续变量：区间估计与两组比较

1.1 均值的置信区间

Step 1 / 肿瘤厚度（thickness）均值的 95% 置信区间
x = mel$thickness
t.test(x)$conf.int

# 同时输出均值与标准误
c(mean = mean(x), se = sd(x) / sqrt(length(x)))
写作模板（示例）：报告连续变量的均值时，建议把区间一起写出来，例如：mean = 2.10，95%CI [1.80, 2.40]（单位与含义按变量解释补全）。

1.2 两组比较：Welch / Wilcoxon

分组变量：用 ulcer（0/1）分组比较 thickness。这是典型“两独立样本、连续结局”的比较场景。
Step 2 / 先看分组描述统计与箱线图
by(mel$thickness, mel$ulcer, summary)

boxplot(thickness ~ factor(ulcer), data = mel,
        xlab = "ulcer (0/1)", ylab = "thickness",
        main = "thickness by ulcer")
Step 3 / Welch t 检验与 Wilcoxon 秩和检验
t.test(thickness ~ ulcer, data = mel)      # Welch（默认）
wilcox.test(thickness ~ ulcer, data = mel) # 非参数
Step 4 / 效应量：差值（以及差值的置信区间）
tt = t.test(thickness ~ ulcer, data = mel)   # Welch
tt$estimate                                 # 两组均值
tt$conf.int                                 # 均值差的 95%CI

# Cohen's d（用合并标准差；仅作量级参考）
x0 = mel$thickness[mel$ulcer == 0]
x1 = mel$thickness[mel$ulcer == 1]
sp = sqrt(((length(x0) - 1) * var(x0) + (length(x1) - 1) * var(x1)) / (length(x0) + length(x1) - 2))
d  = (mean(x1) - mean(x0)) / sp
d
Step 5 / 稳健性：对偏态更友好的变换（可选）
mel$log_thickness = log1p(mel$thickness)
t.test(log_thickness ~ ulcer, data = mel)
wilcox.test(log_thickness ~ ulcer, data = mel)
提醒：不要只盯着 p 值。优先关注“差多少”（差值/效应量）与不确定性（置信区间），并结合图 2 判断差异是否来自少数离群点。

2. 分类变量：列联表与检验

2.1 频数、构成比与风险差

Step 4 / 列联表：性别（sex）× 溃疡（ulcer）
tb = table(ulcer = mel$ulcer, sex = mel$sex)
tb

# 构成比：行百分比（每个 ulcer 组内部）
round(prop.table(tb, 1) * 100, 1)
Step 5 / 风险差（示例）：ulcer=1 在不同性别中的比例差
p_male = mean(mel$ulcer[mel$sex == 0] == 1)
p_fem  = mean(mel$ulcer[mel$sex == 1] == 1)
c(p_male = p_male, p_female = p_fem, diff = p_male - p_fem)
Step 6 / 可视化：不同 sex 的 ulcer 构成比
tb = table(ulcer = mel$ulcer, sex = mel$sex)
prop = prop.table(tb, 2)   # 每个 sex 组内部的构成比
round(prop * 100, 1)

2.2 卡方检验与 Fisher

Step 7 / 卡方检验（含期望频数）
chi = chisq.test(tb)
chi
chi$expected
Step 8 / Fisher 精确检验（2×2 表）
fisher.test(tb)
Step 9 / 效应量：OR 与 RR（2×2 表）
# 把 2×2 表按 (ulcer=1 vs 0) × (sex=1 vs 0) 取出来
tb2 = table(ulcer = mel$ulcer, sex = mel$sex)
tb2

a = tb2["1","1"]; b = tb2["0","1"]
c = tb2["1","0"]; d = tb2["0","0"]

OR = (a * d) / (b * c)
RR = (a / (a + b)) / (c / (c + d))
c(OR = OR, RR = RR)

# Wald 近似 95%CI（对数尺度）
se_log_or = sqrt(1/a + 1/b + 1/c + 1/d)
ci_or = exp(log(OR) + c(-1, 1) * 1.96 * se_log_or)
ci_or
注意：卡方检验依赖“期望频数”条件；当期望频数过小时，优先用 Fisher 精确检验或合并类别后再检验。
