09 · 聚类与降维入门

1 聚类思路与 K-means

聚类是无监督学习：在没有“标签”的情况下，根据变量把样本分成若干组，使组内相似、组间差异大。在医学中可用于亚型发现、人群分层等。常用方法包括 K-means（需事先指定类数 k）和层次聚类（可得到树状图再切分）。

1.1 无监督与距离

聚类依据的是样本间的“距离”（欧氏距离、曼哈顿距离等）；数值变量通常需先标准化，避免量纲影响。

1.2 kmeans() 与解读

# 只取数值列，并标准化
X <- scale(d[, c("age", "dose_g", "score")])
km <- kmeans(X, centers = 3, nstart = 25)
km$cluster   # 每个样本所属类
km$centers   # 各类中心
table(km$cluster)

2 层次聚类简介

层次聚类从“每个样本为一类”开始，逐步合并最相近的两类，得到树状图（dendrogram）；按高度切割即得到类数。R 中可用 hclust + dist。

# 距离矩阵 → 层次聚类 → 树状图
D <- dist(X)
hc <- hclust(D, method = "ward.D2")
plot(hc)
# 切分为 3 类
cutree(hc, k = 3)

3 PCA 概念与 prcomp

主成分分析（PCA）是一种降维方法：用少数几个“主成分”（原变量的线性组合）保留大部分方差，便于可视化和后续分析。医学中常用于多指标综合、共线性处理前的维度压缩。

# 数值矩阵（建议先 scale）
pca <- prcomp(X, scale. = TRUE)
summary(pca)   # 各主成分方差比例
pca$x[, 1:2]   # 前两主成分得分，可做散点图
biplot(pca)    # 双标图（样本 + 变量方向）
