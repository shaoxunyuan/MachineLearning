04 · 统计推断基础

一、连续变量（Continuous Variables）

连续变量通常指有单位、可以无限细分的数据，例如血压、血糖、身高、基因表达量等。

1.1 概念与描述

分析时常先考察分布形态（近似正态与否、方差是否齐性），再选择参数或非参数方法。

1.2 参数估计

点估计与区间估计是推断总体的基本手段。

  • 样本均数 \(\bar{x}\) 与标准差 \(s\)：描述集中趋势与离散程度。
  • 置信区间（CI）：推断总体均数可能所在的范围。例如：“该人群平均收缩压 95% CI 为 120–130 mmHg”。

1.3 假设检验（比较与关联）

根据组数与分布特征选择检验：

  • 两组比较：近似正态、方差齐性可满足时，用独立样本 \(t\) 检验；非正态或方差不齐时，用 Mann–Whitney \(U\) 检验（非参数）。
  • 多组比较：正态与方差分析前提满足时，用方差分析（ANOVA）；非正态时，用 Kruskal–Wallis 检验。

1.4 变量关联

  • Pearson 相关：探讨两个连续变量之间线性相关的强弱。
  • 线性回归推断：建立 \(Y = \beta_0 + \beta_1 X\) 模型，并对斜率 \(\beta_1\) 等进行统计推断。

二、分类变量（Categorical Variables）

分类变量为定性数据，例如性别（二分类）、病情严重程度（等级）、血型（多分类）等。

2.1 概念与描述

需区分名义分类与有序分类（等级），后者在检验与解释上与名义变量不完全相同。

2.2 参数估计

  • 率 \(p\)与构成比：描述各类别出现的频率。
  • 率的置信区间：推断总体患病率、有效率等。

2.3 假设检验（频率的比较）

  • 两组/多组率的比较：卡方检验（\(\chi^2\)）最常用，例如比较 A、B 两组手术成功率。
  • Fisher 确切概率法：样本量很小或期望频数过低时使用。
  • 等级资料：可用秩和检验等，例如比较“无效、好转、显效”在不同药物组间的分布。

2.4 关联分析

  • 比值比（OR）与相对危险度（RR）：医学数据挖掘中衡量风险的核心指标。
  • Logistic 回归推断：当因变量为分类结局（如患病/未患病）时，对影响因素的显著性进行推断。

三、混合情况：连续 vs 分类

在数据挖掘与回归建模中，常同时涉及不同类型自变量与因变量的组合，可按下表选择常用的统计推断或模型思路。

3.1 自变量与因变量组合

自变量（X） 因变量（Y） 推荐统计推断 / 模型方法
分类（如：组别） 连续（如：身高） \(t\) 检验 / ANOVA
连续（如：年龄） 分类（如：生存） Logistic 回归
分类（如：性别） 分类（如：疗效） 卡方检验 / 列联表分析
连续（如：剂量） 连续（如：反应） 线性回归 / 相关分析

四、针对数据挖掘的特殊推断

当变量很多、分析步骤复杂（高维表格、影像或信号特征、反复建模等）时，经典“一个假设、一次检验”的框架不够用，需要额外约束整体错误率与结论的可重复性。

4.1 多重假设检验校正

若对大量特征各自做显著性检验而不校正，偶然出现的“显著”结果会成倍增加。除 Bonferroni 等较保守的方法外，高通量场景下还常用 FDR（False Discovery Rate） 控制在一批检验中假阳性所占比例的期望上界，便于在探索阶段平衡发现与可靠性。

4.2 验证划分与可重复性

在同一批数据上先筛选变量、调模型再报告 p 值，往往使推断过于乐观。更稳妥的做法是：训练 / 验证（或测试）划分、嵌套交叉验证等，使“选谁进模型”与“评估好坏”尽量用到不同数据；重要结论宜在外部独立队列或前瞻性数据上复核。探索性分析中得到的模式，宜在后续研究中按预先注册的假设再做验证。对流程很复杂的情形，还可借助置换检验等判断输出是否明显优于随机打乱标签下的结果。

4.3 因果推断与倾向得分匹配（PSM）

定位：因果推断（含 PSM 等方法）常被概括为医学数据挖掘从“描述现状”走向“指导治疗与实践决策”的最高阶段：要回答的是“若采取某干预，结局可能如何”，而不仅是“变量之间是否相关”。达到这一阶段需要在观察性数据中审慎处理混杂、明确识别假设，因而也是整个流程里最难做扎实的一环。

随机对照试验（RCT）通过随机化使混杂在期望意义上平衡，因果解释相对直接。真实世界数据多为观察性研究：治疗组与对照组往往在基线特征上不可比，此时关联不等于因果。倾向得分（Propensity Score）是在给定协变量条件下接受某处理的概率估计；倾向得分匹配（PSM）、倾向得分加权、逆概率加权（IPW）等，是在可测混杂已充分调整、无重要未测混杂与模型正确设定等假设下，构造可比子总体、估计处理效应的常用工具。

因果推断依赖明确的研究问题、有向无环图或类似框架厘清混杂与中介，并配合敏感性分析。其结论力度仍通常弱于设计良好的 RCT，但能在伦理或可行性限制下为治疗选择与资源分配提供补充证据。
