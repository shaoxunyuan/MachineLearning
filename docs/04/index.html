04 · 统计推断基础

一、统计推断在做什么

统计推断的目标是从样本对总体做结论，例如“两组疗效是否不同”“某指标总体均值是否等于某参考值”。推断的核心难点是：我们永远拿不到总体的全部信息，因此必须面对抽样带来的不确定性。

1.1 总体、样本与不确定性

总体（Population）
研究真正关心的对象集合，例如“某地区 2 型糖尿病患者”。
样本（Sample）
从总体抽取的有限观测，决定了估计的误差大小。
随机误差
同一总体反复抽样，样本统计量会波动，这就是抽样不确定性。
系统偏倚
选择偏倚、信息偏倚、混杂等不会被“多抽点样本”自动消除。
医学解读：统计推断处理的是“随机误差”；但研究质量更常被“系统偏倚”左右，因此推断应结合研究设计与数据质量。

1.2 参数、统计量与抽样分布

  • 参数：总体特征（如总体均值 μ、总体差异 Δ）。
  • 统计量：由样本计算得到（如样本均值 \(\bar{x}\)、均值差 \(\bar{x}_1-\bar{x}_2\)）。
  • 抽样分布：统计量在重复抽样下的分布，是 p 值与置信区间的基础。

二、假设检验与 p 值

假设检验先设定原假设 H₀（通常为“无差异/无效应”），再计算在 H₀ 成立时得到当前或更极端数据的概率，即 p 值。p 很小表示数据与 H₀ 不相容，可拒绝 H₀。

2.1 H₀ / H₁ 与单双侧

原假设 H₀
如：两组均值相等（μ₁ = μ₂）；或均值差为 0（Δ = 0）。
备择假设 H₁
如：μ₁ ≠ μ₂（双侧），或 μ₁ > μ₂ / μ₁ < μ₂（单侧）。

单侧/双侧应在分析前确定。医学研究中通常优先双侧，除非存在明确的方向性科学假设。

2.2 Ⅰ类/Ⅱ类错误与效能

  • Ⅰ类错误（假阳性）：H₀ 真的却拒绝了 H₀，概率为 α（常设 0.05）。
  • Ⅱ类错误（假阴性）：H₀ 假的却未拒绝 H₀，概率为 β。
  • 检验效能（Power）：\(1-\beta\)，发现真实差异的能力，受样本量、方差、效应大小影响。
常见误区：p>0.05 并不等于“没有差异”，也可能是样本量太小导致效能不足。

2.3 p 值的正确解读

  • p 值不是“原假设为真的概率”。
  • p 值不是“结果为偶然的概率”。
  • p 值不反映效应大小；小效应在大样本中也可能 p 很小。

更好的报告方式是：效应量 + 置信区间 + p 值，并解释临床意义。

三、t 检验与替代方法

t 检验用于比较均值：单样本、两独立样本、配对样本。要求数据近似正态或样本量较大；若严重偏态可考虑非参数检验。

3.1 单样本 / 两独立样本

示例可用 R 内置数据集 ToothGrowth（维生素 C 对牙齿生长的影响）：比较不同补充方式（supp）在某剂量下的差异。

data(ToothGrowth)
head(ToothGrowth)

# 只取剂量 0.5，比较两组均值（两独立样本）
d05 <- subset(ToothGrowth, dose == 0.5)
t.test(len ~ supp, data = d05, var.equal = FALSE)  # Welch t 检验
Welch vs 等方差：两组方差不齐时优先 Welch（默认）；只有在能证明方差齐时才用等方差版本。

3.2 配对 t 检验

配对设计用于“同一对象前后比较”或“匹配对照”，能减少个体差异带来的方差。

# 构造一个简单的前后数据示例
before <- c(120, 130, 128, 115, 140)
after  <- c(118, 125, 126, 112, 136)
t.test(before, after, paired = TRUE)

3.3 非参数检验

当数据严重偏态、存在大量离群点，或为等级数据时，可考虑非参数方法（基于秩）：

  • Wilcoxon 秩和检验：两独立样本（对应 Mann–Whitney U）。
  • Wilcoxon 符号秩检验：配对样本。
# 两独立样本非参数检验
wilcox.test(len ~ supp, data = d05)

# 配对非参数检验
wilcox.test(before, after, paired = TRUE)

四、分类变量检验

分类变量之间的关联（如证型与疗效是否有关）通常从二维列联表出发。

4.1 卡方检验

卡方检验用于判断“行变量与列变量是否独立”。适用于样本量较大、期望频数不太小的情形。

# R 内置 Titanic 数据：舱位与生存是否相关？
data(Titanic)
tb <- margin.table(Titanic, margin = c(1,4))  # Class × Survived
tb
chisq.test(tb)

4.2 Fisher 精确检验

当期望频数过小（常见规则：任一格 < 5）时，卡方近似不可靠，可用 Fisher 精确检验（适用于 2×2 或小表）。

# 2×2 表示例（可用你自己的数据替换）
tb2 <- matrix(c(1, 9, 11, 3), nrow = 2, byrow = TRUE)
fisher.test(tb2)

五、置信区间与效应量

推断的目标不仅是“有没有差异”，更重要的是“差异有多大，是否有临床意义”。因此建议把置信区间与效应量放在核心位置。

5.1 置信区间的意义

  • 95% CI 可以理解为：在相同方案下重复抽样，95% 的区间会覆盖真实参数。
  • CI 的宽窄反映估计的精度：样本量越大，区间通常越窄。
  • 对“差值”的 CI：若 CI 不包含 0，通常与 p<0.05 一致。
# t.test 输出中直接包含 conf.int
out <- t.test(len ~ supp, data = d05)
out$estimate
out$conf.int

5.2 效应量优先于 p 值

为什么要看效应量
p 值受样本量影响很大；效应量能更直接表达“差异大小”。
常见效应量
均值差、标准化均值差（Cohen's d）、风险差、相对风险、OR 等。
# 一个简单的 Cohen's d（两独立样本）
x <- d05$len[d05$supp == "OJ"]
y <- d05$len[d05$supp == "VC"]
d <- (mean(x) - mean(y)) / sqrt((var(x) + var(y)) / 2)
d
