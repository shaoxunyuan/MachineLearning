04 · 探索性数据分析与描述统计

1 探索性分析思路

探索性数据分析（EDA）是在正式建模或假设检验之前，用汇总统计和图形“摸清”数据的分布、关联和异常，为后续分析选模型、选变量、写报告提供依据。医学数据中，不同证型的人数分布、年龄与疗效的关系、各组的均值与标准差等，都属于描述统计范畴。

1.1 EDA 的目的

目的包括：了解每个变量的分布形态（是否偏态、有无离群点）、了解变量间是否存在明显关联或分层差异、检查是否还有未处理的异常或缺失、为论文或报告中的“表 1 基线特征”提供数值与图表。

1.2 描述与推断的区分

描述统计只对“当前样本”做概括，不涉及总体推断（如均值、标准差、频数、百分位数）；推断统计则用假设检验或置信区间对“总体”下结论。本章重点在描述：如何用 R 算出一组数的均值与分位数、如何按组汇总、如何用表格和图形呈现。

2 单变量描述

单变量描述是 EDA 的基础：对数值变量用中心趋势与离散程度（均值、中位数、标准差、分位数），对分类变量用频数与比例。

2.1 数值：均值与分位数

# 单列：均值、标准差、中位数、分位数
mean(d$age, na.rm = TRUE)
sd(d$age, na.rm = TRUE)
median(d$dose_g, na.rm = TRUE)
quantile(d$dose_g, probs = c(0.25, 0.5, 0.75), na.rm = TRUE)

# 一次看多列摘要
summary(d[, c("age", "dose_g")])

2.2 分类：频数与比例

# 一维频数表
table(d$zheng)

# 比例（相对频数）
prop.table(table(d$zheng))

# 带缺失的计数
table(d$zheng, useNA = "ifany")

3 分组汇总与交叉表

医学分析常需要“按证型/分组/性别”看各组的均值、人数等。用 dplyr 的 group_by + summarise 可灵活实现；两个分类变量之间的交叉用 table 或 xtabs。

3.1 group_by + summarise

library(dplyr)

# 按证型分组，算每组的年龄均值、剂量均值、人数
d %>% group_by(zheng) %>%
  summarise(
    n = n(),
    age_mean = mean(age, na.rm = TRUE),
    dose_mean = mean(dose_g, na.rm = TRUE)
  )

3.2 table 与交叉表

# 两分类变量交叉表（如证型 × 性别）
table(d$zheng, d$sex)

# 带边际的交叉表
addmargins(table(d$zheng, d$sex))

4 常用可视化

描述统计的结果常配合图形呈现：分类变量用条形图、数值变量用直方图或箱线图、两变量关系用散点图或分组箱线图。这些在第二章已用 ggplot2 入门，此处可结合分组汇总结果画“按证型的剂量箱线图”“按组的年龄直方图”等，用于报告或汇报。

library(ggplot2)

# 按证型分组的剂量箱线图
ggplot(d, aes(x = zheng, y = dose_g, fill = zheng)) +
  geom_boxplot(show.legend = FALSE) +
  labs(x = "证型", y = "剂量 (g)") +
  theme_minimal()

5 小结与练习

本节要求：能对数值列计算均值、标准差、分位数，对分类列做频数表与比例；能使用 group_by + summarise 做分组汇总，用 table 做交叉表；能结合 ggplot2 画出描述性图表并解读。

要点：EDA 以描述为主；mean、sd、quantile、table、group_by + summarise 与基础图形是核心工具。
