08 · 数据整合与变形

1 长表与宽表

同一份数据可以有“宽”和“长”两种形式：宽表是“每个观测一行、多个指标各占一列”（如患者 ID + 第1次血压 + 第2次血压）；长表是“每次测量一行”（ID + 时间点 + 血压值）。很多 R 函数（如 ggplot2、混合效应模型）更适合长表；而录入或导出常是宽表。需要在两者之间转换并合并多张表。

2 pivot_longer / pivot_wider

tidyr 的 pivot_longer 把多列“收拢”成“名—值”两列（宽转长）；pivot_wider 把“名—值”展开成多列（长转宽）。

2.1 宽转长

library(tidyr)

# 宽表：id, bp_1, bp_2, bp_3 → 长表：id, visit, bp
d_long <- pivot_longer(d, cols = starts_with("bp_"),
  names_to = "visit", values_to = "bp",
  names_prefix = "bp_")

2.2 长转宽

# 长表：id, visit, bp → 宽表：id, visit_1, visit_2, ...
d_wide <- pivot_wider(d_long, names_from = visit, values_from = bp,
  names_prefix = "visit_")

3 表合并：join

多张表按“键”列（如患者 ID）合并：left_join 保留左表所有行并匹配右表；inner_join 只保留两表都有的键；full_join 保留全部。来自 dplyr。

library(dplyr)

# 按 id 合并：保留 d1 全部行，把 d2 的列贴上去
d_merged <- left_join(d1, d2, by = "id")

# 键名不同时
left_join(d1, d2, by = c("patient_id" = "id"))
