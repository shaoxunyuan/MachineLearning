三种常用类型

  • 数值：剂量、年龄、化验值
  • 字符：药名、证型、方剂名
  • 逻辑：TRUE / FALSE，用于筛选

变量与赋值

dose_g  <- 30      # 数值
herb    <- "黄芪"  # 字符
revisit <- TRUE    # 逻辑

用 class() 或 typeof() 查看类型

向量与 data.frame

  • 向量：一列同类型数据，c(10, 12, 9)
  • 数据框：多列、行=观测，像 Excel 表

医学数据最常用 data.frame：每行一条记录，每列一个变量。
