03 · 描述性统计与数据概览

一、描述性统计与数据概览

在做假设检验和建模之前，第一步总是：先弄清楚手里的数据“长什么样”。描述性统计（descriptive statistics）就是围绕三个问题展开：

  • 中心在哪里？——集中趋势
  • 差异有多大？——离散程度
  • 形状是什么样？——分布形状

本章以 R 内置数据集 iris 和 mtcars 为例演示，代码可以直接在 RStudio 里复制运行。

1.1 为什么要先“描述”再建模

防止“黑箱”分析
  • 直接套用检验/模型，可能在严重偏态、极端值很多的数据上得出“看起来显著”的错误结论。
  • 描述性统计帮你发现：中心在哪里、数据是否合理、是否存在可疑值或分层结构。
为后续决策提供依据
  • 决定用均值±标准差还是中位数（IQR）。
  • 决定用 t 检验还是非参数检验。
  • 决定是否需要对数变换或标准化。

1.2 变量类型与合适的指标

不同类型的变量，适合的描述指标不同：

连续型（数值）
年龄、检验值等：通常用均值/中位数、标准差/IQR、直方图、箱线图描述。
有序分类
分级评分（轻/中/重）：不适合直接算均值，可以用中位数、分布比例、条形图。
无序分类
性别、证型、地区等：用频数/比例、条形图/饼图描述。
时间序列
多次测量或监护数据：需要考虑时间顺序，用折线图、滑动平均等方法概览。

二、描述数据的“集中趋势”

集中趋势描述“数据大致集中在哪个水平”。常用的三个指标是：平均数、中位数、众数。

2.1 平均数、中位数、众数

  • 平均数（mean）：所有观测值相加除以个数，容易理解，但对极端值异常敏感。
  • 中位数（median）：排序后正中间的值，对偏态和极端值更稳健。
  • 众数（mode）：出现次数最多的取值，常用于分类变量。
# 示例：iris 数据集
data(iris)
x <- iris$Sepal.Length

mean(x)    # 平均数
median(x)  # 中位数

# 众数：适用于分类变量，这里示例 Species
get_mode <- function(v) {
  tb <- table(v)
  names(tb)[which.max(tb)]
}

get_mode(iris$Species)

2.2 医学场景下如何报告

在医学论文中，常见的写法有：

  • 近似正态的连续变量：报告“均值 ± 标准差”（例如：年龄 56.2±8.4 岁）。
  • 偏态分布的连续变量：报告“中位数（IQR）”（例如：住院天数 7（4, 12）天）。
  • 分类变量：报告“频数（比例）”（例如：男性 60 例（55.6%））。
# 用 iris 构造类似“表 1”行
mean_SL  <- mean(x);  sd_SL  <- sd(x)
med_SL   <- median(x); iqr_SL <- IQR(x)

sprintf("Sepal.Length: %.2f ± %.2f", mean_SL, sd_SL)
sprintf("Sepal.Length: %.2f (%.2f, %.2f)",
        med_SL,
        quantile(x, 0.25),
        quantile(x, 0.75))

三、描述数据的“离散程度”

离散程度回答“数据之间差异有多大”。同样的平均值，离散程度不同，意味着临床上可能完全不同的稳定性。

3.1 极差、方差与标准差

  • 极差（Range）：最大值 − 最小值，直观但受极端值影响极大。
  • 方差（Variance）：每个值相对平均数的平方偏差的平均。
  • 标准差（SD）：方差的平方根，单位与原变量相同，更易解释。
max(x) - min(x)    # 极差
var(x)             # 方差
sd(x)              # 标准差

3.2 四分位距与变异系数

  • 四分位距（IQR）：Q3 − Q1，反映中间 50% 数据范围，比极差稳健。
  • 变异系数（CV）：\(CV = sd/mean\)，用于比较不同量纲下的相对离散程度。
IQR(x)
q <- quantile(x, c(0.25, 0.5, 0.75))
q

cv <- sd(x) / mean(x)
cv

3.3 医学例子中的“波动解读”

同样的平均值，标准差不同，在医学上的含义可能截然不同：

  • 血压均值 130 mmHg，标准差 5 vs 25：前者更稳定，后者可能提示血压控制不良或测量不规范。
  • 住院天数中位数 7 天，IQR 为 2–30 天：说明有一部分患者住院极长，需要单独分析原因。

四、描述数据的“分布形状”

分布形状不仅影响描述方式，还决定后续推断方法能否使用（例如 t 检验更适合近似正态的情形）。

4.1 偏度与峰度

  • 偏度（Skewness）：衡量分布的不对称性，右偏/左偏。
  • 峰度（Kurtosis）：衡量分布的尖峭程度与尾部厚度。
skewness <- function(v) {
  v <- v[is.finite(v)]
  m <- mean(v); s <- sd(v)
  mean((v - m)^3) / (s^3)
}

kurtosis <- function(v) {
  v <- v[is.finite(v)]
  m <- mean(v); s <- sd(v)
  mean((v - m)^4) / (s^4)
}

skewness(x)
kurtosis(x)

4.2 频数/频率分布与分箱

对连续变量，可以通过分箱得到频数/频率表：

bins <- cut(x, breaks = 8)
freq <- table(bins)
freq
prop.table(freq)

分箱数量过少，细节会丢失；过多则会显得杂乱。一般 8–15 个箱是常见的起点，具体需结合样本量调整。

4.3 分布形状与后续方法选择

  • 近似对称、单峰：可考虑使用均值±SD、t 检验等参数方法。
  • 明显偏态、长尾、多峰：更适合报告中位数（IQR），考虑对数变换或非参数方法。
  • 多峰分布：可能提示混合人群（例如不同疾病亚型），可尝试分层分析或聚类。

五、常用图表

数值指标回答“多少”，图形回答“长什么样”。两者结合，才能在汇报和论文中讲清数据。

5.1 直方图与密度曲线

hist(x, breaks = 15,
     main = "Sepal.Length 直方图",
     xlab  = "Sepal.Length")
lines(density(x), col = "red", lwd = 2)

5.2 箱线图与异常值

箱线图展示五数概括（最小值、Q1、中位数、Q3、最大值），并把超过“须”（通常是 Q1−1.5×IQR、Q3+1.5×IQR）之外的点标记为潜在异常值。

boxplot(x, main = "Sepal.Length 箱线图")

5.3 散点图与关系探索

plot(iris$Sepal.Length, iris$Petal.Length,
     xlab = "Sepal.Length",
     ylab = "Petal.Length",
     main = "散点图：Sepal vs Petal")

5.4 条形图 / 饼图（分类数据）

tb <- table(iris$Species)

# 条形图
barplot(tb, main = "Species 条形图")

# 饼图
pie(tb, main = "Species 饼图")

六、小结与练习

本章从“中心”“差异”“形状”三个角度系统介绍了描述性统计，并配合直方图、箱线图、散点图、条形图/饼图等图表，构成了完整的数据概览工具箱。

练习建议（可用 R 内置数据集）：
  • 选取 iris 中任意一个数值变量，计算其均值/中位数/极差/SD/IQR，并画直方图和箱线图；用 2–3 句话描述它的“中心、差异、形状”。
  • 用 mtcars 比较自动/手动（am）两组的 mpg：分别画直方图和箱线图，并讨论哪一组的油耗更稳定。
  • 任选一个研究问题（例如“不同性别的某指标是否不同”），写出你会在“结果”部分如何用描述性统计 + 图表来呈现基线情况。

03 · 探索性数据分析与描述统计

本章定位：本页按“描述性统计”的典型大纲展开：集中趋势 → 离散程度 → 分布形状 → 常用图表。示例主要使用 R 内置数据集（如 iris、mtcars），便于直接复现。

1. 描述数据的“集中趋势”

集中趋势回答“数据中心在哪里”。最常用的三个指标是：平均数、中位数、众数。

平均数（Mean）
所有数值加起来除以个数。优点是容易理解，缺点是容易受极端值影响。
中位数（Median）
排序后位于正中间的数。更稳健，更能代表“普通水平”。
众数（Mode）
出现次数最多的数。常用于分类数据（例如最常见的证型、最受欢迎品牌）。
一个小建议
偏态或有离群点时，更推荐报告 中位数（IQR） 而不是均值±标准差。
# 示例数据：iris（内置）
data(iris)
x <- iris$Sepal.Length

# 平均数 / 中位数
mean(x)
median(x)

# 众数（通用写法：数值/字符/因子都可）
get_mode <- function(v) {
  tb <- table(v)
  names(tb)[which.max(tb)]
}

get_mode(iris$Species)  # 分类数据的众数

2. 描述数据的“离散程度”

离散程度回答“数据之间的差异有多大”。常用指标有：极差、方差/标准差、四分位距。

  • 极差（Range / 全距）：最大值 − 最小值。简单但对极端值敏感。
  • 方差与标准差（Var / SD）：衡量数据围绕平均数的波动，SD 单位与原数据一致，更直观。
  • 四分位距（IQR）：\(IQR = Q3 - Q1\)，反映中间 50% 数据的范围，更稳健。
# 仍以 iris 的 Sepal.Length 为例
max(x) - min(x)   # 极差
var(x)            # 方差
sd(x)             # 标准差
IQR(x)            # 四分位距

quantile(x, probs = c(0.25, 0.5, 0.75))  # Q1 / 中位数 / Q3

3. 描述数据的“分布形状”

分布形状回答“数据长什么样”。常见关注点包括：偏度、峰度，以及频数/频率分布。

偏度（Skewness）
衡量分布的对称性。右侧长尾称右偏；左侧长尾称左偏。
峰度（Kurtosis）
衡量分布“陡峭/平坦”的程度（与尾部厚度也有关）。
频数与频率
统计每个值或区间出现次数，构建频率分布表可快速了解分布。
实践建议
比起单看偏度/峰度，更建议结合直方图/箱线图一起判断。
# 偏度/峰度：用“中心矩”的基础实现（不依赖额外包）
skewness <- function(v) {
  v <- v[is.finite(v)]
  m <- mean(v); s <- sd(v)
  mean((v - m)^3) / (s^3)
}

kurtosis <- function(v) {
  v <- v[is.finite(v)]
  m <- mean(v); s <- sd(v)
  mean((v - m)^4) / (s^4)
}

skewness(x)
kurtosis(x)

# 频数/频率分布表（按区间分箱）
bins <- cut(x, breaks = 8)
freq <- table(bins)
freq
prop.table(freq)

4. 常用图表

图表是描述性统计最直观的表达方式。

4.1 直方图

用于查看整体分布形态（例如是否近似正态、是否长尾）。

hist(x, breaks = 12, main = "Sepal.Length 直方图", xlab = "Sepal.Length")

4.2 箱线图

一次展示最小值、Q1、中位数、Q3、最大值，并能标识潜在异常值。

boxplot(x, main = "Sepal.Length 箱线图")

4.3 散点图

用于探索两个变量之间的关系（线性/非线性、离群点、分层等）。

plot(iris$Sepal.Length, iris$Petal.Length,
     xlab = "Sepal.Length", ylab = "Petal.Length",
     main = "散点图：两变量关系")

4.4 条形图 / 饼图（分类数据）

适用于描述性别、地区等分类数据。类别很多时更推荐条形图，饼图适合类别很少、比例差异明显的情况。

# 条形图：分类频数
tb <- table(iris$Species)
barplot(tb, main = "条形图：Species 频数")

# 饼图：分类比例
pie(tb, main = "饼图：Species 构成")
