03 · 探索性数据分析与描述统计

一、探索性数据分析（EDA）

探索性数据分析（Exploratory Data Analysis, EDA）是正式建模前的“体检”：用描述统计与可视化快速理解数据结构、发现异常与模式、明确后续分析策略。医学数据的 EDA 还需要结合临床合理范围与业务口径，避免“看似异常其实合理”的误判。

1.1 目标与产出

你要回答的问题
  • 样本量有多少？关键变量缺失/异常是否严重？
  • 连续变量的分布如何（偏态/长尾）？分类变量的构成如何？
  • 分组（如性别/治疗组）之间是否存在明显差异？
  • 变量间关系是否线性？是否存在离群点/分层现象？
你应该输出的东西
  • 数据字典与口径说明（单位、编码、时间窗）
  • 核心变量的描述表（均值/中位数/四分位数/缺失率）
  • 关键图形（直方图、箱线图、散点图、分组柱状图）
  • 问题清单：缺失机制猜测、异常值规则、需补充数据的字段
提示：EDA 的目标不是“证明假设”，而是帮助你形成可检验的假设并规划后续检验/建模。

1.2 变量类型与分析路径

选择统计量与图形前，先明确变量类型：

连续型（数值）
年龄、剂量、检验值。常看分布、均值/中位数、方差/IQR、异常值。
分类型（类别）
性别、证型、结局。常看频数/比例、条形图、交叉表。
有序分类
分级（轻/中/重）、疼痛评分等级。注意顺序信息，常用中位数、秩和方法。
时间与时序
入院/出院、随访时间点、监护波形。需统一时间尺度，常用趋势图/区间统计。

二、描述统计

描述统计回答“数据大概长什么样”。医学场景中常同时报告均值±标准差与中位数（IQR），以适配正态与偏态两类分布。

2.1 集中趋势

  • 均值（mean）：对极端值敏感，适合近似正态的连续变量。
  • 中位数（median）：对偏态与离群点更稳健。
  • 众数（mode）：常用于分类变量。
# R 内置数据集示例：iris
data(iris)

mean(iris$Sepal.Length)
median(iris$Sepal.Length)

# 分类变量频数
table(iris$Species)

2.2 离散程度

  • 标准差（sd）与方差（var）：刻画波动大小。
  • 四分位距（IQR）：偏态分布更常用。
  • 变异系数（CV）：\(CV = sd/mean\)，便于不同量纲变量比较（均值接近 0 时慎用）。
sd(iris$Sepal.Length)
IQR(iris$Sepal.Length)

cv <- sd(iris$Sepal.Length) / mean(iris$Sepal.Length)
cv

2.3 分布与偏态

分布形态决定后续方法选择：

  • 近似正态：均值±标准差、t 检验、线性模型更常见。
  • 明显偏态/长尾：中位数（IQR）、对数变换、非参数检验更常见。
# 直方图与密度（基础图形）
hist(iris$Sepal.Length, breaks = 15, main = "Sepal.Length", xlab = "")
lines(density(iris$Sepal.Length), col = "red", lwd = 2)

三、分组汇总与交叉表

医学研究常关心“分组差异”：比如不同治疗组、不同性别、不同证型的指标差异。分组汇总是后续检验与建模的前置步骤。

3.1 分组描述

建议同时输出每组的样本量、缺失数、均值/中位数、IQR 等。

# iris：按物种分组汇总
aggregate(Sepal.Length ~ Species, data = iris, FUN = mean)
aggregate(Sepal.Length ~ Species, data = iris, FUN = median)

# dplyr 方式（更常用）
library(dplyr)
iris %>%
  group_by(Species) %>%
  summarise(
    n = n(),
    mean_SL = mean(Sepal.Length),
    sd_SL = sd(Sepal.Length),
    med_SL = median(Sepal.Length),
    iqr_SL = IQR(Sepal.Length)
  )

3.2 交叉表与比例

对分类变量，用交叉表看构成差异，并计算行/列比例。

# mtcars：变速箱 am (0/1) 与汽缸数 cyl 的交叉表
data(mtcars)
tb <- table(cyl = mtcars$cyl, am = mtcars$am)
tb

# 行比例（每种 cyl 内，am 的比例）
prop.table(tb, margin = 1)
提示：交叉表是卡方/Fisher 检验的输入，也是混杂识别（分层差异）的直观工具。

四、可视化基础

图形能比表格更快暴露问题：偏态、离群点、异方差、分层结构等。建议先用基础图形完成 EDA，再按需要升级到 ggplot2 体系。

4.1 单变量图形

连续变量
直方图/密度图看分布；箱线图看离群点与四分位。
分类变量
条形图看频数或比例；类别太多时建议合并低频。
# 箱线图：Sepal.Length
boxplot(iris$Sepal.Length, main = "Sepal.Length 箱线图")

4.2 两变量关系

  • 连续×连续：散点图（必要时加趋势线），看线性/非线性与离群点。
  • 连续×分类：分组箱线图/小提琴图，看组间差异与离群点。
  • 分类×分类：分组柱状图或马赛克图，看构成差异。
# 连续×连续：散点图
plot(iris$Sepal.Length, iris$Petal.Length,
     xlab = "Sepal.Length", ylab = "Petal.Length")

# 连续×分类：分组箱线图
boxplot(Sepal.Length ~ Species, data = iris, main = "按 Species 分组")

4.3 分组比较（一个小套路）

当你要比较“不同人群的某指标”时，建议按以下顺序：

  1. 先看每组样本量与缺失（避免小样本与选择偏倚）。
  2. 画分组箱线图/散点图（先看分布与离群点）。
  3. 再做统计检验/建模（下一章再讲）。

五、小结与练习

本节要求：能用描述统计与基础图形完成一轮 EDA；能按分组输出汇总表与交叉表；能从图形中识别偏态、离群点与分组差异，为后续推断与建模做准备。

练习建议（可用 R 内置数据集）：
  • 用 iris 做：按 Species 汇总 4 个数值变量的均值/中位数/IQR，并画分组箱线图。
  • 用 mtcars 做：比较自动/手动（am）两组的 mpg 分布（直方图 + 箱线图）。
  • 尝试写一段“EDA 结论段落”：说明样本量、异常值、分布形态、主要差异点。
