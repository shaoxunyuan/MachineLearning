目录

本系列面向医学/中医药数据分析入门：从 R 语言基础与数据类型，到数据质量评估、清洗、描述统计，再到统计推断、线性/逻辑回归、数据整合、聚类降维，最后以可复现分析与 R Markdown 收尾。左侧点击章节进入正文；建议按 01 → 10 顺序学习。

建议学习路径：01 → 02 → 03 → 04 → 05 → 06 → 07 → 08 → 09 → 10
