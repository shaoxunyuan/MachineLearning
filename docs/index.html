课程定位

本课程面向希望系统学习医学数据处理的同学，以「数据—模型—解释」为主线， 覆盖电子病历与真实世界数据、中药处方与配伍挖掘、以及基因表达与功能富集三类典型场景。 不涉及 R / RStudio 的安装与项目配置说明，该部分统一在仓库 README 中维护。

三大方向

01

临床与真实世界数据

基于模拟 EHR 与真实世界数据结构，完成质量评估、缺失与异常处理、编码与特征构建，支撑后续预测与因果推断。

02

中药与方剂挖掘

从处方数据出发，运用关联规则与网络方法识别核心药对与配伍模式，并与中医理论相结合做结果解读。

03

组学与生物信息

处理基因表达矩阵与差异分析结果，开展 GO / KEGG 等富集分析，并完成从统计结果到生物学含义的转化与可视化。

你将获得

  • 使用 R 与 tidyverse 完成医学数据清洗、转换与质量评估的完整流程
  • 医学统计建模、关联规则与差异/富集分析的基本原理与实现
  • 从原始数据到可解释结论的建模与报告能力（含 R Markdown）
