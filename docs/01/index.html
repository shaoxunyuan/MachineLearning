01 · R 语言数据类型与常用运算（零基础版）

这一节只干一件事：看懂“数据是什么类型”，并知道怎么对它做运算。 你不需要背概念，跟着敲代码就行。

最重要的 3 句话：
  • R 里向量是最基础的结构：c(1,2,3)。
  • 同一个向量里尽量放同一种类型；混着放会被“强制转换”。
  • 遇到奇怪结果先用：class(x)、typeof(x)、str(x) 检查类型。

0. 先准备：创建变量 + 查看类型

x <- 10
y <- 3.14
z <- "hello"
b <- TRUE

class(x)   # 类（更贴近“你该怎么用它”）
typeof(x)  # 底层类型（更贴近“它在内存里是什么”）
str(x)     # 一眼看结构（非常常用）

1. 数值类型：整数（integer）与小数（double）

R 默认把 1 当作 double（小数类型），要整数需要写 1L（L 表示 integer）。

创建
a <- 1
class(a)      # "numeric"（通常是 double）

i <- 1L
class(i)      # "integer"
四则运算
5 + 2
5 - 2
5 * 2
5 / 2     # 除法
5 %% 2    # 取余数
5 %/% 2   # 整除（向下取整）

常见坑：小数精度

0.1 + 0.2
# 可能不是精确的 0.3（计算机用二进制存小数）

all.equal(0.1 + 0.2, 0.3)  # 用它判断更稳

2. 逻辑类型：TRUE / FALSE 与比较运算

逻辑值通常来自“比较”。比较出来的是 TRUE 或 FALSE。

3 > 2
3 >= 2
3 < 2
3 == 2     # 等于
3 != 2     # 不等于

# 逻辑运算
TRUE & FALSE   # 且（AND）
TRUE | FALSE   # 或（OR）
!TRUE          # 非（NOT）

& 和 && 的区别（考试常考，但你现在只记一句）

& 对向量逐个比较；&& 只看第一个元素（多用于 if 判断）。

c(TRUE, FALSE) & c(TRUE, TRUE)    # 得到长度为 2 的结果
c(TRUE, FALSE) && c(TRUE, TRUE)   # 只看第一个：TRUE && TRUE

3. 字符类型：字符串（character）

字符串一定要用引号：\"abc\"。常见操作：拼接、长度、取子串、替换。

s <- "MedData"
nchar(s)                 # 字符长度
paste("hello", s)        # 拼接（默认有空格）
paste0("hello", s)       # 拼接（不加空格）
substr(s, 1, 3)          # 取子串：第 1 到 3 个字符

gsub("Data", "DATA", s)  # 替换

字符转数值

as.numeric("12")   # 12
as.numeric("12a")  # NA（转不了，会变 NA，并给 warning）

4. 因子类型：factor（分类变量）

因子适合“类别”，例如性别、分组、分型。它有“水平”（levels）。

sex <- factor(c("男","女","女","男"))
sex
levels(sex)
table(sex)   # 频数统计
提醒：因子不是字符串。很多时候你需要先 as.character() 再处理。

5. 缺失值：NA / NaN / Inf（医学数据里非常常见）

x <- c(1, NA, 3)
is.na(x)          # 哪些是 NA
sum(is.na(x))     # 缺失个数

mean(x)           # 结果是 NA（因为包含 NA）
mean(x, na.rm=TRUE)  # 忽略缺失

NA 的比较：不要用 ==

NA == NA        # 还是 NA（不是 TRUE/FALSE）
is.na(NA)       # TRUE（正确写法）

6. 向量（vector）：R 最重要的数据结构

向量就是“一串同类型的数据”。你最常用的是 c()、索引、向量化运算。

v <- c(10, 20, 30, 40)
v[1]       # 第 1 个
v[2:3]     # 第 2 到 3 个
v[c(1,4)]  # 取第 1 和 4 个

# 向量化运算：一次对整串做运算
v + 1
v / 10
v > 25

混合类型会发生“强制转换”

c(1, 2, "3")
class(c(1, 2, "3"))  # 变成 character

7. 常见容器快速认识：list / matrix / data.frame

你现在只要知道它们分别“像什么”。后面课程会反复用到。

7.1 list：可以装不同类型（“杂货箱”）

lst <- list(a=1, b="x", c=c(1,2,3))
lst$a
lst[[2]]

7.2 matrix：二维，所有元素同类型（“表格，但全是同一种”）

m <- matrix(1:6, nrow=2)
m
m[1, 2]

7.3 data.frame：二维表格，不同列可以不同类型（医学数据最常见）

df <- data.frame(
  id = 1:3,
  age = c(18, 19, 20),
  sex = factor(c("男","女","女")),
  stringsAsFactors = FALSE
)

df
df$age
df[1, ]
df[, c("id","sex")]

8. 练习（建议你真的敲一遍）

  1. 创建向量 x <- c(3, 6, 9, NA, 12)，求它的平均值（忽略 NA）。
  2. 创建字符串 s <- \"2026-03-06\"，尝试转成日期：as.Date(s)。
  3. 创建因子 g <- factor(c(\"A\",\"B\",\"A\",\"C\"))，输出各类别频数。
  4. 创建数据框 df（至少 3 列，含数值/字符/因子），分别用 df$col 和 df[,\"col\"] 取出一列。
下一节建议：如果你已经能分清 vector、data.frame、list， 并且知道 NA 要用 is.na() 来判断，那你就已经超过大多数零基础同学了。
