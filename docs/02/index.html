02 · 数据质量评估与可视化

1 数据质量概述

在正式做统计建模或数据挖掘之前，必须先了解数据“长什么样、缺多少、有没有明显错误”。数据质量评估就是在不改变原始数据的前提下，用汇总统计和图形把缺失、异常、分布、变量间关系等摸清楚，为后续的清洗和建模提供依据。医学数据尤其如此：录入错误、随访缺失、单位不统一等问题很常见，先评估再动手能避免无效分析。

1.1 为什么先评估

盲目清洗或建模容易掩盖问题或引入新偏差。先看 summary()、缺失比例、简单分布图，能帮助决定：哪些变量值得保留、缺失该删还是该补、异常值是否合理（如血压 200 是否笔误）。

1.2 常见问题类型

典型问题包括：缺失（NA、空字符串、未填写）、异常值（超出生理或测量范围）、重复（同一条记录多行）、类型错误（本应是数值却存成字符）、单位或编码不统一（如性别有时“男/女”有时“1/2”）。评估阶段先识别，清洗阶段再处理。

2 缺失与异常初探

用 R 读入数据后，第一步通常是看整体结构和各列摘要；再针对缺失和关键数值列做定量与可视化检查。

2.1 summary 与结构

# 假设已读入数据框 d（如从 CSV）
# 整体结构：行数、列数、每列类型
str(d)

# 数值列：最小、四分位、均值、最大、NA 数
summary(d)

# 只看缺失个数
colSums(is.na(d))

2.2 缺失模式与可视化

了解“哪些变量缺、缺多少”后，可用条形图或热力图展示缺失比例；数值变量用箱线图或直方图看分布和离群点。

# 缺失比例（每列）
pct_na <- colSums(is.na(d)) / nrow(d) * 100
barplot(pct_na, las = 2, main = "各列缺失比例(%)")

# 若已装 ggplot2：某数值变量分布
library(ggplot2)
ggplot(d, aes(x = 1, y = dose_g)) + geom_boxplot(outlier.alpha = 0.6)

3 基础可视化

可视化是质量评估和探索分析的核心手段。R 中 ggplot2 是当前最常用的图形系统：用“图层 + 美学映射”的方式画分布、关系、分组对比等，便于发现异常和模式。

3.1 ggplot2 入门

基本思路：ggplot(数据, aes(x = ..., y = ..., color = ...)) 定义画布与映射，再用 + geom_xxx() 指定几何类型（点、线、条、箱线等）。

library(ggplot2)

# 示例：证型频数条形图（zheng 为因子列）
ggplot(d, aes(x = zheng)) + geom_bar(fill = "steelblue") +
  labs(x = "证型", y = "频数", title = "证型分布") +
  theme_minimal()

3.2 分布与关系图

单变量分布常用直方图或密度图；两变量关系常用散点图；分组对比可用箱线图或分组条形图。这些图既能用于质量评估（看异常、偏态），也可用于后续探索性分析。

# 数值变量直方图
ggplot(d, aes(x = age)) + geom_histogram(bins = 20, fill = "steelblue", alpha = 0.8)

# 两数值变量散点图（可加 color = zheng 按证型着色）
ggplot(d, aes(x = age, y = dose_g)) + geom_point(alpha = 0.6) + theme_minimal()

4 小结与练习

本节要求：能对读入的数据做 str / summary 与缺失统计，并能用 ggplot2 画出条形图、箱线图、直方图、散点图，用于初步评估数据质量和分布。

要点：数据质量评估先于清洗与建模；summary、is.na、colSums(is.na(...)) 与基础图形是必备工具。
