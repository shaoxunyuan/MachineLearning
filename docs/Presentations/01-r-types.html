1 变量与类型检查

变量 = 给数据贴的标签。中医药数据里：剂量、药名、是否复诊、证型 等都会存成不同类型。先会创建变量、会用函数查类型，是后面一切分析的起点。

1.1 创建变量

# 剂量：数值
dose_g <- 10

# 药名：字符
herb <- "黄芪"

# 是否复诊：逻辑
revisit <- TRUE

1.2 类型查看函数

class(dose_g)
typeof(dose_g)
str(dose_g)

class(herb)
class(revisit)

2 数值：剂量与运算

剂量、体温、血压、疗程天数等。R 里数值以向量形式存在；熟悉整数/小数、四则/取余/幂运算是剂量计算和统计量的基础。

2.1 整数与小数

x <- 1
class(x)     # "numeric"（double）

i <- 1L
class(i)     # "integer"

2.2 常用运算（剂量/疗程）

huangqi_g <- 30
times_per_day <- 2
each_time_g <- huangqi_g / times_per_day

days <- 15
days %/% 7   # 完整疗程数
days %% 7    # 剩余天数

2.3 向量化运算

doses <- c(15, 10, 6, 12, 3)
doses + 1
doses / sum(doses)
R 的运算支持向量化，可对整组数据逐元素计算。

3 逻辑：筛选条件

TRUE / FALSE / NA，表示“是否”类信息。比较与逻辑运算得到一列 TRUE/FALSE，配合下标即可按条件取子集。

3.1 比较运算 · 3.2 & 与 &&

age <- c(18, 20, 19, 22)
age >= 20

# & 逐元素；&& 只看第一个（常用于 if）
c(TRUE, FALSE) & c(TRUE, TRUE)
c(TRUE, FALSE) && c(TRUE, TRUE)

4 字符：药名与文本

中药名、方剂名、证型描述等。可做拼接、替换、截取；配合正则与 stringr 可从病案中提取药名、剂量等。

4.1 拼接与长度 · 4.2 替换与截取

herbs <- c("黄芪", "当归", "党参")
nchar(herbs)
paste(herbs, "10g")
paste0(herbs, "_10g")

s <- "当归 10g"
gsub("10g", "12g", s)
substr(s, 1, 2)

4.3 字符转数值

as.numeric("12")     # 12
as.numeric("12g")    # NA

5 因子：证型分类

表示“类别”：性别、证型、分组等。带 levels，统计函数会按水平做分组或虚拟变量。

5.1 levels 与频数 · 5.2 因子坑

zheng <- factor(c("气虚", "血虚", "气虚", "湿热", "气虚"))
levels(zheng)
table(zheng)

# 文本处理建议先 as.character
z <- factor(c("A", "B", "A"))
paste0("组别=", as.character(z))

6 缺失值：NA

真实数据常有缺失。有 NA 时 mean、sum 等默认得 NA；先用 is.na() 找缺失，再用 na.rm = TRUE 等处理。

6.1 is.na · 6.2 na.rm

doses2 <- c(10, NA, 6, 12)
is.na(doses2)
sum(is.na(doses2))

mean(doses2)                 # NA
mean(doses2, na.rm = TRUE)   # 正常
判断缺失用 is.na()；NA == NA 结果仍是 NA。

7 容器：vector / list / data.frame

vector = 一列同类型；list = 一个病例的全部信息；data.frame = 行×列表格，最常用。

7.1 vector · 7.2 list

dose_col <- c(10, 12, 9, 15)

patient <- list(
  id = "P001", age = 19, zheng = "气虚",
  herbs = c("黄芪", "党参", "白术"),
  doses = c(30, 15, 12)
)
patient$id
patient$herbs

7.3 data.frame（行=观测，列=变量）

rx <- data.frame(
  patient_id = c("P001", "P002", "P003"),
  zheng = factor(c("气虚", "湿热", "血虚")),
  formula = c("补中益气汤", "三仁汤", "四物汤"),
  herb = c("黄芪", "薏苡仁", "当归"),
  dose_g = c(30, 20, 10),
  stringsAsFactors = FALSE
)
rx$dose_g
rx[1, ]
rx[, c("patient_id", "formula", "dose_g")]

  3. 因子 z <- factor(c("气虚","血虚","气虚"))，输出频数。
  4. 在 rx 里筛出 dose_g >= 20 的行。
要点：区分 vector / data.frame / list，掌握 is.na() 与向量化运算。
