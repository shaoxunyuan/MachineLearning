1 数据质量分析概述

数据预处理的前提，检查原始数据中是否存在"脏数据"。

1.1 什么是数据质量分析

在不对原始数据做任何修改的前提下，通过统计方法和可视化手段全面了解数据：

  • 完整性检查：识别缺失值及其分布模式
  • 准确性检查：发现异常值和错误数据
  • 一致性检查：检查数据格式、单位、编码是否统一
  • 合理性检查：验证数据是否符合业务逻辑和医学常识

1.2 脏数据的类型

缺失数据

字段值为空（NA、NULL），如患者年龄未填写

异常数据

超出正常范围，如血压 300 mmHg、年龄为负数

重复数据

同一条记录出现多次，如同一患者重复录入

不一致数据

格式不统一，如性别用"男/女"或"1/2"

2 缺失值分析

医学数据中最常见的质量问题，了解原因、影响及检测方法。

2.1 缺失值产生的原因

  • 信息难以获取：检查费用昂贵或具有侵入性
  • 人为遗漏：录入疏忽或患者拒绝回答敏感问题
  • 属性值不存在：某些特征不适用于特定患者
  • 设备故障：检验设备故障导致无法测量
  • 数据整合问题：多源数据合并时字段不匹配

2.2 缺失值的影响

  • 统计量偏差：含缺失值的数据计算均值、方差会得到错误结果
  • 样本量减少：删除缺失记录会降低统计检验效能
  • 模型失真：处理不当会引入偏差，影响模型准确性
  • 结论不可靠：基于不完整数据的结论可能无法反映真实情况

2.3 缺失值检测函数

# 创建示例数据
d <- data.frame(
  id = c(1, 2, 3, 4, 5),
  age = c(25, NA, 30, 45, NA),
  bp = c(120, 130, NA, 140, 125)
)

# is.na() - 返回TRUE/FALSE矩阵
is.na(d)

# 统计每列缺失值个数
colSums(is.na(d))
#  id age  bp 
#   0   2   1

# complete.cases() - 返回逻辑向量
complete.cases(d)
# [1]  TRUE FALSE FALSE FALSE FALSE

2.4 缺失值可视化

# 条形图展示缺失比例
pct_na <- colSums(is.na(d)) / nrow(d) * 100
barplot(pct_na, main = "各列缺失比例(%)")

# 使用mice包查看缺失模式
library(mice)
md.pattern(d)

# 使用VIM包可视化缺失值
library(VIM)
aggr(d, col = c('navyblue', 'red'),
     numbers = TRUE, sortVars = TRUE)
提示 可视化有助于发现缺失值的分布模式，指导后续处理策略。

3 异常值分析

识别明显偏离其他观测值的数据点，可能是录入错误或真实极端病例。

3.1 异常值的含义与来源

  • 录入错误：人工录入时的笔误，如"120"录成"1200"
  • 测量误差：仪器故障或操作不当导致的错误读数
  • 真实极端值：患者确实存在极端的生理指标
  • 数据整合错误：不同数据源的单位不一致

3.2 简单统计量分析

# 创建示例数据（包含异常值）
values <- c(23, 25, 22, 24, 26, 23, 25, 150, 24, 23)

# 使用summary()查看基本统计量
summary(values)
#    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
#   22.00   23.00   24.00   36.50   25.25  150.00

# 计算Z分数（标准化值）
z_scores <- scale(values)

# 识别|Z分数|>3的异常值
outliers <- abs(z_scores) > 3
values[outliers]
# [1] 150
观察：最大值150明显偏离其他值，Z分数约为2.96。

3.3 3σ原则

基于正态分布的性质：约99.7%的数据落在均值±3倍标准差范围内。

# 3σ原则识别异常值
values <- c(23, 25, 22, 24, 26, 23, 25, 150, 24, 23)

# 计算均值和标准差
mean_val <- mean(values)
sd_val <- sd(values)

# 确定上下界
lower_bound <- mean_val - 3 * sd_val
upper_bound <- mean_val + 3 * sd_val

# 识别异常值
outliers <- values < lower_bound | values > upper_bound
cat("异常值:", values[outliers], "\n")
# 异常值: 150
注意 3σ原则假设数据近似正态分布，偏态数据需使用其他方法。

3.4 箱型图分析

基于四分位数，不受极端值影响，适用于各种分布形态。

# 基础箱型图
boxplot(values, main = "箱型图检测异常值")

# 使用ggplot2绘制分组箱型图
library(ggplot2)

data <- data.frame(
  group = rep(c("A", "B", "C"), each = 200),
  value = c(rnorm(200, 0, 1),
            rnorm(200, 3, 2),
            rnorm(200, 6, 1.5))
)

ggplot(data, aes(x = group, y = value, fill = group)) +
  geom_boxplot(outlier.colour = "red") +
  theme_minimal()

箱型图异常值判定规则

IQR（四分位距） = Q3 - Q1

异常值：小于 Q1 - 1.5×IQR 或 大于 Q3 + 1.5×IQR

极端异常值：小于 Q1 - 3×IQR 或 大于 Q3 + 3×IQR

箱型图的优点：不依赖正态分布假设，对偏态数据更稳健。

3.5 质量控制图

医学检验和工业生产中监控过程稳定性的重要工具。

# 使用qcc包绘制质量控制图
library(qcc)

# 使用内置的活塞环数据集
data(pistonrings)
attach(pistonrings)

# 绘制单值控制图
qcc(diameter, type = "xbar.one",
    title = "活塞环直径质量控制图")

# 模拟新数据并绘制控制图
set.seed(123)
newdata <- rep(74, 30) + rnorm(30) / 50
qcc(diameter, type = "xbar.one", newdata = newdata)

质量控制图解读

  • 中心线（CL）：过程的平均值
  • 上控制限（UCL）：均值 + 3σ
  • 下控制限（LCL）：均值 - 3σ
  • 失控信号：超出控制限的点表示过程失控
医学应用：监控检验指标的稳定性，如血糖试纸的测量准确性。

小结 · 速查

  • 数据质量分析是数据预处理的前提
  • 脏数据类型：缺失、异常、重复、不一致
  • 缺失值检测：is.na()、complete.cases()
  • 缺失值可视化：mice::md.pattern()、VIM::aggr()
  • 3σ原则：超出均值±3σ视为异常值
  • 箱型图：基于IQR识别异常值
  • 质量控制图：qcc包监控过程稳定性
