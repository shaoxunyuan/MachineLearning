1 数据质量分析概述

数据预处理的前提，检查原始数据中是否存在"脏数据"。

1.1 什么是数据质量分析

在不对原始数据做任何修改的前提下，通过统计方法和可视化手段全面了解数据：

完整性检查

识别缺失值及其分布模式

准确性检查

发现异常值和错误数据

一致性检查

检查数据格式、单位、编码是否统一

合理性检查

验证数据是否符合业务逻辑和医学常识

1.2 脏数据的类型

类型 描述 医学数据示例
缺失数据 字段值为空（NA、NULL） 患者年龄未填写、检验结果缺失
异常数据 超出正常范围 血压 300 mmHg、年龄为负数
重复数据 同一条记录出现多次 同一患者的相同检查记录被重复录入
不一致数据 格式、单位或编码不统一 性别用"男/女"或"1/2"

2 数据清洗原理

去重、格式转换、时间格式统一

2.1 去重处理

识别并删除重复记录的两种策略：

完全重复

所有字段值都相同的记录

保留第一条，删除后续重复记录

部分重复

关键字段（如ID）相同，其他字段可能不同

根据业务规则合并或选择最新记录

3 缺失值分析

理解缺失机制，选择恰当的处理策略

3.1 缺失值机制

MCAR - 完全随机缺失

Missing Completely At Random

缺失与任何变量（包括自身）都无关

例：随机抽血的样本丢失

MAR - 随机缺失

Missing At Random

缺失与其他观测变量有关，但与缺失值本身无关

例：老年人更易缺失血压数据

MNAR - 非随机缺失

Missing Not At Random

缺失与缺失值本身有关

例：病情严重者更可能缺失随访

3.2 缺失值可视化模式

图2：缺失模式热力图示意（绿色=有数据，红色=缺失）

3.3 缺失值处理策略

策略 适用场景 优缺点
删除法 缺失比例低（<5%），MCAR机制 简单，但会减少样本量
均值/中位数填充 数值型变量，缺失比例中等 简单，但会降低方差
回归插补 有相关变量可预测缺失值 更准确，但会过度拟合
多重插补 缺失比例高，需要准确估计 最严谨，但计算复杂

4 描述性统计分析

通过基本统计量了解数据分布特征

4.1 集中趋势度量

算术平均数（Mean）

x̄ = (Σxi) / n = (x₁ + x₂ + ... + xn) / n

所有观测值的平均值，受异常值影响较大

加权平均数

x̄w = Σ(wi · xi) / Σwi

中位数（Median）

M = x((n+1)/2) （n为奇数）
M = (x(n/2) + x(n/2+1)) / 2 （n为偶数）

将数据排序后位于中间位置的值，对异常值稳健

4.2 离散程度度量

方差（Variance）

s² = Σ(xi - x̄)² / (n-1)

标准差（SD）

s = √s² = √[Σ(xi - x̄)² / (n-1)]

四分位距（IQR）

IQR = Q₃ - Q₁

第三四分位数与第一四分位数之差

变异系数（CV）

CV = (s / x̄) × 100%

用于比较不同量纲数据的离散程度

5 异常值分析

识别明显偏离其他观测值的数据点

5.1 3σ原则（拉依达准则）

基于正态分布的性质：约99.7%的数据落在均值±3倍标准差范围内

下界：LCL = x̄ - 3s
上界：UCL = x̄ + 3s
图4：3σ原则示意图（超出μ±3σ范围的值视为异常值）

5.2 箱型图分析

箱型图组成

判定规则

IQR = Q₃ - Q₁

异常值界限：

下限 = Q₁ - 1.5 × IQR
上限 = Q₃ + 1.5 × IQR

极端异常值界限：

下限 = Q₁ - 3 × IQR
上限 = Q₃ + 3 × IQR

5.3 质量控制图

图5：质量控制图示意（超出UCL/LCL的点表示过程失控）
控制限含义：
CL（中心线）= 过程平均值
UCL（上控制限）= μ + 3σ
LCL（下控制限）= μ - 3σ

6 相关性分析

研究变量之间的关系强度和方向

6.1 Pearson相关系数

衡量两个连续变量之间的线性相关程度

r = Σ(xi - x̄)(yi - ȳ) / √[Σ(xi - x̄)² · Σ(yi - ȳ)²]

取值范围：-1 ≤ r ≤ 1

|r| 范围 相关程度
0.8 ~ 1.0 极强相关
0.6 ~ 0.8 强相关
0.4 ~ 0.6 中等相关
0.2 ~ 0.4 弱相关
0.0 ~ 0.2 极弱相关或无相关
适用条件 连续变量、线性关系、近似正态分布

6.2 Spearman相关系数

基于秩次的非参数相关度量

rs = 1 - [6 × Σ(Ri - Qi)²] / [n(n² - 1)]

其中 Ri 和 Qi 分别是X和Y的秩次

图6：Spearman相关系数先将原始数据转换为秩次
适用场景 有序变量、非线性单调关系、存在异常值

6.3 三种相关系数比较

相关系数 数据类型 关系类型 对异常值敏感度
Pearson 连续变量 线性 敏感
Spearman 连续/有序 单调 较稳健
Kendall 连续/有序 单调 最稳健
选择建议：
• 数据正态分布且无线性关系 → Pearson
• 存在异常值或非线性单调 → Spearman
• 小样本或需要最稳健结果 → Kendall

7 数据插补原理

根据已知数据点估计缺失值

7.1 线性插值

假设相邻数据点之间呈线性关系

y = y₁ + (x - x₁) × (y₂ - y₁) / (x₂ - x₁)
图7：线性插值原理（通过两点确定直线，估计中间缺失值）

7.2 样条插值

使用分段多项式，产生更平滑的插值曲线

线性插值

折线连接，转折明显

样条插值

平滑曲线，更符合自然规律

三次样条插值：在每两个相邻点之间使用三次多项式，保证曲线平滑且通过所有已知点

8 数据合并

整合来自多个数据源的数据

8.1 合并类型

行合并（纵向）

将多个数据集按行堆叠

要求：列结构相同

列合并（横向）

将多个数据集按列拼接

要求：行数相同

内连接

只保留两表都有的记录

结果：交集

左/右连接

保留左表/右表所有记录

结果：缺失值填充NA
