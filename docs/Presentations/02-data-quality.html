1 数据质量分析概述

数据预处理的前提，检查原始数据中是否存在"脏数据"。

1.1 什么是数据质量分析

在不对原始数据做任何修改的前提下，通过统计方法和可视化手段全面了解数据：

  • 完整性检查：识别缺失值及其分布模式
  • 准确性检查：发现异常值和错误数据
  • 一致性检查：检查数据格式、单位、编码是否统一
  • 合理性检查：验证数据是否符合业务逻辑和医学常识

1.2 脏数据的类型

缺失数据

字段值为空（NA、NULL），如患者年龄未填写

异常数据

超出正常范围，如血压 300 mmHg、年龄为负数

重复数据

同一条记录出现多次，如同一患者重复录入

不一致数据

格式不统一，如性别用"男/女"或"1/2"

2 数据清洗

去重、格式转换、时间格式统一。

2.1 去重处理

# 查看重复行
duplicated(df)

# 删除完全重复的行
df_unique <- df[!duplicated(df), ]

# 根据特定列去重
df %>% distinct(id, .keep_all = TRUE)

2.2 格式转换

# 字符型转数值型
df$age <- as.numeric(df$age)

# 字符型转因子型
df$gender <- factor(df$gender, 
                    levels = c("1", "2"), 
                    labels = c("男", "女"))

# 统一日期格式
df$date <- as.Date(df$date_str, format = "%Y-%m-%d")

2.3 时间格式统一

library(lubridate)

# 解析各种日期时间格式
time_parsed <- ymd_hms(time_string)  # 标准格式
time_parsed <- ymd(time_string)      # 日期格式

# 提取时间成分
year(time_parsed)
month(time_parsed)
day(time_parsed)
hour(time_parsed)

3 缺失值分析

医学数据中最常见的质量问题。

3.1 缺失值机制

MCAR - 完全随机缺失

缺失与任何变量都无关。如随机抽血的样本丢失。

MAR - 随机缺失

缺失与其他观测变量有关。如老年人更易缺失血压数据。

MNAR - 非随机缺失

缺失与缺失值本身有关。如病情严重者更可能缺失随访。

3.2 缺失值检测

# 判断是否为缺失值
is.na(d)

# 统计每列缺失值个数
colSums(is.na(d))

# 查看缺失模式
library(mice)
md.pattern(d)

# 可视化缺失值
library(VIM)
aggr(d, col = c('navyblue', 'red'))

3.3 缺失值处理

# 删除含有缺失值的行
df_clean <- na.omit(df)

# 均值/中位数填充
df$x[is.na(df$x)] <- mean(df$x, na.rm = TRUE)

# 多重插补
library(mice)
imputed_data <- mice(df, m = 5, method = "pmm")
complete_data <- complete(imputed_data, 1)

4 描述性统计分析

通过基本统计量了解数据分布特征。

4.1 集中趋势度量

# 算术平均数（受异常值影响）
mean(values, na.rm = TRUE)

# 加权平均数
weighted.mean(values, weights)

# 中位数（对异常值稳健）
median(values, na.rm = TRUE)

# 分位数
quantile(values, probs = c(0.25, 0.5, 0.75))

4.2 离散程度度量

# 极差
max(data) - min(data)

# 四分位距
IQR(data)

# 方差和标准差
var(data)
sd(data)

# 变异系数（用于比较不同量纲数据）
cv <- (sd(data) / mean(data)) * 100

4.3 分布形态分析

library(moments)

# 偏度：>0右偏，<0左偏，=0对称
skewness(data)

# 峰度：>3尖峰，<3平峰，=3正态
kurtosis(data)

# 正态性检验
shapiro.test(data)

5 异常值分析

识别明显偏离其他观测值的数据点。

5.1 3σ原则

# 计算上下界
lower_bound <- mean(values) - 3 * sd(values)
upper_bound <- mean(values) + 3 * sd(values)

# 识别异常值
outliers <- values < lower_bound | values > upper_bound

# 注意：假设数据近似正态分布
注意 3σ原则假设数据近似正态分布，偏态数据使用IQR方法。

5.2 箱型图分析

# 基础箱型图
boxplot(values, main = "箱型图检测异常值")

# 使用ggplot2绘制分组箱型图
ggplot(data, aes(x = group, y = value)) +
  geom_boxplot(outlier.colour = "red")

箱型图判定规则

IQR（四分位距） = Q3 - Q1

异常值：小于 Q1 - 1.5×IQR 或 大于 Q3 + 1.5×IQR

极端异常值：小于 Q1 - 3×IQR 或 大于 Q3 + 3×IQR

5.3 质量控制图

library(qcc)

# 绘制单值控制图
qcc(diameter, type = "xbar.one",
    title = "质量控制图")

# 控制限解读：
# - 中心线（CL）：过程的平均值
# - 上控制限（UCL）：均值 + 3σ
# - 下控制限（LCL）：均值 - 3σ

6 相关性分析

研究变量之间的关系强度和方向。

6.1 Pearson相关系数

衡量两个连续变量之间的线性相关程度

# 计算Pearson相关系数
cor(x, y, method = "pearson")

# 相关矩阵
cor(data, method = "pearson")

# 相关性检验
cor.test(x, y, method = "pearson")
|r|>0.8极强相关，0.6-0.8强相关，0.4-0.6中等，0.2-0.4弱，<0.2极弱

6.2 Spearman相关系数

基于秩次，适用于非线性关系或有序变量

# Spearman相关系数
cor(x, y, method = "spearman")

# 适用场景：
# - 数据不满足正态分布
# - 存在异常值
# - 变量为有序分类变量
# - 关系为非线性但单调

6.3 Kendall相关系数

基于秩次，对异常值更稳健

# Kendall相关系数
cor(x, y, method = "kendall")

# 选择建议：
# - Pearson：连续变量，线性关系，正态分布
# - Spearman：有序变量，非线性单调关系
# - Kendall：小样本，需要更稳健的结果

8 数据合并

整合来自多个数据源的数据。

8.1 行合并与列合并

# 行合并
rbind(df1, df2)
bind_rows(df1, df2)

# 列合并
cbind(df1, df2)
bind_cols(df1, df2)

8.2 主键合并

library(dplyr)

# 内连接
inner_join(df1, df2, by = "patient_id")

# 左连接
left_join(df1, df2, by = "patient_id")

# 全连接
full_join(df1, df2, by = "patient_id")

# 多主键合并
merge(df1, df2, by = c("patient_id", "visit_no"))
