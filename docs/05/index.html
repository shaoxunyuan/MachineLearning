05 · 统计推断基础

1 假设检验与 p 值

统计推断是从样本对总体做结论：例如“两组疗效是否有差异”“某指标总体均值是否等于某值”。假设检验先设定“原假设 H₀”（如两总体均值相等），再根据样本数据计算在 H₀ 成立时出现当前或更极端结果的概率，即 p 值；p 很小则拒绝 H₀。医学论文中 p < 0.05 常被用作“有统计学意义”的阈值，但解读时需结合效应量和临床意义。

1.1 原假设与备择假设

原假设通常为“无差异、无效应”（如 μ₁ = μ₂）；备择假设为“有差异”（双侧）或“某侧更大”（单侧）。检验前应明确假设与检验类型（t 检验、卡方等）及是否单侧。

1.2 p 值的含义

p 值是在原假设为真的前提下，得到当前样本或更极端结果的概率。p 小说明当前数据与原假设不太相容，可拒绝 H₀；p 大则不拒绝，但“不拒绝”不等于“证明 H₀ 成立”。

2 t 检验

t 检验用于比较均值：单样本（总体均值是否等于某常数）、两独立样本（两组均值是否相等）、配对样本（前后或配对观测之差是否为零）。要求数据近似正态或样本量较大；若严重偏态可考虑非参数方法（如 Wilcoxon）。

2.1 单样本与两独立样本

# 单样本 t 检验：某列是否与总体均值 0 不同（或 mu = 某值）
t.test(d$score, mu = 0)

# 两独立样本 t 检验：按 group 分两组，比较某指标均值
t.test(value ~ group, data = d, var.equal = FALSE)  # Welch 不假定方差齐

2.2 配对 t 检验

# 配对：治疗前后同一批患者的指标
t.test(d$before, d$after, paired = TRUE)

3 卡方与 Fisher 检验

分类变量之间的关联（如证型与疗效是否有关）常用 卡方检验（chisq.test）；当期望频数过小（如任一格 < 5）时可用 Fisher 精确检验（fisher.test）。

# 二维列联表：证型 × 疗效
tb <- table(d$zheng, d$outcome)
chisq.test(tb)

# 小样本或期望频数小时
fisher.test(tb)

4 置信区间

除 p 值外，应报告效应量的置信区间（如均数差的 95% CI），便于判断临床意义。R 中 t.test、wilcox.test 等会直接给出区间。

# t.test 输出中包含 mean difference 的 conf.int
out <- t.test(value ~ group, data = d)
out$conf.int   # 均数差的 95% CI

5 小结与练习

本节要求：理解假设检验与 p 值的含义；能根据数据类型选用 t 检验或卡方/Fisher 检验并在 R 中执行；能读取并报告置信区间。

要点：t.test（单样本/两样本/配对）、chisq.test、fisher.test 及结果中的 p.value、conf.int 是基础工具。
