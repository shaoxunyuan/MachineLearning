05 · 线性回归与模型诊断

1 简单线性回归

线性回归用于刻画一个连续型因变量与一个或多个自变量之间的线性关系，在医学中常用于“剂量–效应”“年龄–指标”等。简单线性回归只有一个自变量：y = β₀ + β₁x + ε，用最小二乘估计 β，并得到 R²、残差等。

1.1 模型与 lm()

# 因变量 score，自变量 age
fit <- lm(score ~ age, data = d)
summary(fit)

1.2 系数与 R²

summary(fit) 给出截距与斜率的估计、标准误、t 检验、p 值，以及 R²（决定系数）和调整 R²。系数解读为“x 每增加 1 单位，y 平均变化 β₁ 单位”。

2 多重线性回归

加入多个自变量可控制混杂、考察多个因素对结局的独立贡献。模型形式为 y = β₀ + β₁x₁ + … + βₚxₚ + ε；分类自变量需转为因子（R 会自动处理为虚拟变量）。

# 多自变量：数值 + 因子
fit2 <- lm(score ~ age + sex + dose_g, data = d)
summary(fit2)

# 提取系数、置信区间
coef(fit2)
confint(fit2)

3 残差与模型诊断

线性回归假定：线性、残差正态、方差齐性、观测独立。可通过残差图、Q-Q 图、残差 vs 拟合值图等做简单诊断；plot(fit) 可一次生成多张诊断图。

# 四张诊断图（顺序：残差 vs 拟合、Q-Q、尺度-位置、残差 vs 杠杆）
plot(fit, which = 1:4)

4 小结与练习

本节要求：能用 lm() 拟合简单与多重线性回归，读懂 summary 中的系数与 R²；会看残差诊断图并理解其含义。

要点：lm(y ~ x, data)、summary、coef、confint、plot(fit) 是回归分析的基础。
