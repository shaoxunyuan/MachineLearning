07 · 逻辑回归与分类评估

1 二分类与 logit

当因变量为二分类（如有效/无效、发病/未发病）时，线性回归不再适用。逻辑回归用 logit(p) = log(p/(1−p)) 与自变量建立线性关系，得到“某因素对发生概率的影响”的估计，结果常以优势比（OR）表示。

2 glm 与系数解读

# 二分类因变量 y（0/1 或 因子），family = binomial
fit <- glm(outcome ~ age + sex + dose_g, data = d, family = binomial)
summary(fit)

# 系数为 log(OR)，取指数得 OR
exp(coef(fit))
exp(confint(fit))   # OR 的置信区间

系数 β 表示“x 每增加 1 单位，log(OR) 增加 β”；exp(β) 为 OR，即优势比。

3 ROC 与混淆矩阵

分类模型需评估预测效果：混淆矩阵给出真阳/假阳/真阴/假阴；灵敏度、特异度、准确率可由其计算。ROC 曲线以“1−特异度”为横轴、灵敏度为纵轴，曲线下面积 AUC 越接近 1 判别能力越强。

3.1 混淆矩阵与准确率

# 预测概率与 0.5 阈值得到类别
pred_prob <- predict(fit, type = "response")
pred_class <- ifelse(pred_prob >= 0.5, 1, 0)
table(真实 = d$outcome, 预测 = pred_class)

3.2 ROC 曲线与 AUC

# 需 pROC 或 ROCR 等包
library(pROC)
roc_obj <- roc(d$outcome, pred_prob)
plot(roc_obj)
auc(roc_obj)

4 小结与练习

本节要求：能用 glm(..., family = binomial) 拟合逻辑回归，会解读 OR 与置信区间；能计算混淆矩阵并理解 ROC/AUC 的含义。

要点：glm、predict(..., type = "response")、table、pROC::roc / auc 是二分类建模与评估的基础。
